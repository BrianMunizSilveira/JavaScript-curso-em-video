Exercícios com For em JavaScript

1. Soma dos Números de 1 a 100

Escreva um programa que utilize um laço `for` para somar todos os números de 1 a 100. No final, o programa deve exibir a soma.

2. Sequência de Fibonacci

Escreva um programa que peça ao usuário um número e exiba a tabuada desse número de 1 a 10, usando um laço for.

3. Números Pares de 1 a 50

Escreva um programa que exiba todos os números pares entre 1 e 50.

4. Soma dos Números Ímpares

Escreva um programa que calcule a soma de todos os números ímpares de 1 a um número fornecido pelo usuário.

5. Fatorial de um Número

Escreva um programa que calcule o fatorial de um número inteiro positivo, usando um laço for.

6. Reverso de uma Palavra

Escreva um programa que peça uma palavra e exiba a versão invertida dessa palavra usando um laço for.

7. Contagem Regressiva

Escreva um programa que exiba uma contagem regressiva de um número fornecido pelo usuário até 0.

8. Contar Letras de uma String

Escreva um programa que conte quantas vezes uma letra específica aparece em uma string fornecida pelo usuário.

9. Números em Ordem Decrescente

Escreva um programa que exiba os números de 1 até um número fornecido pelo usuário em ordem decrescente.

10. Tabela de Multiplicação de 1 a N

Escreva um programa que exiba a tabela de multiplicação de todos os números de 1 até um número fornecido pelo usuário.
