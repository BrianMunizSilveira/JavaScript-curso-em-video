Entendendo o DOM (Document Object Model)

O que significa a sigla DOM?

DOM significa Document Object Model. É uma interface de programação para documentos HTML e XML que representa a página de forma que os programas possam modificar a estrutura, estilo e conteúdo do documento.

Para que serve o Document Object Model (DOM)?

O DOM serve como uma representação estruturada do documento (como uma árvore de nós) que pode ser acessada e manipulada por scripts (como JavaScript). Permite que os desenvolvedores interajam com a página, alterando dinamicamente o conteúdo, estrutura e estilo do documento sem a necessidade de recarregar a página.

Como criar uma árvore DOM para o seu site?

Quando o navegador carrega um documento HTML, ele cria automaticamente uma árvore DOM. Cada elemento HTML é convertido em um nó na árvore. Por exemplo, uma estrutura simples de HTML:

<html>
  <body>
    <h1>Meu Título</h1>
    <p>Meu parágrafo.</p>
  </body>
</html>

Seria representada na árvore DOM com o nó raiz <html>, que tem um filho <body>, que por sua vez tem filhos <h1> e <p>.

Como usar o JavaScript para manipular o DOM?

JavaScript pode ser usado para manipular o DOM de várias maneiras:

  • Alterar o conteúdo de elementos: document.getElementById('meuId').innerHTML = 'Novo Conteúdo';
  • Modificar atributos de elementos: document.getElementById('meuId').setAttribute('class', 'novaClasse');
  • Adicionar ou remover elementos: document.createElement('div') e element.appendChild(novoElemento)

Para que servem os elementos Parent e Child em um DOM?

No DOM, os elementos Parent (pais) e Child (filhos) referem-se às relações hierárquicas entre os nós da árvore:

  • Parent: É um nó que tem um ou mais nós filhos. Por exemplo, <body> é o parent de <h1> e <p>.
  • Child: É um nó que está subordinado a outro nó. Por exemplo, <h1> e <p> são filhos de <body>.

Como utilizar os métodos de acesso ao DOM no JavaScript?

  • getElementById(id): Seleciona um elemento pelo seu atributo id. Exemplo:
    let elemento = document.getElementById('meuId');
  • getElementsByTagName(tag): Seleciona todos os elementos com o nome da tag especificada. Retorna uma coleção de elementos. Exemplo:
    let paragrafos = document.getElementsByTagName('p');
  • getElementsByName(name): Seleciona todos os elementos com o atributo name especificado. Retorna uma coleção de elementos. Exemplo:
    let elementos = document.getElementsByName('meuNome');
  • getElementsByClassName(class): Seleciona todos os elementos com a classe especificada. Retorna uma coleção de elementos. Exemplo:
    let elementos = document.getElementsByClassName('minhaClasse');
  • querySelector(seletor): Seleciona o primeiro elemento que corresponde ao seletor CSS especificado. Exemplo:
    let elemento = document.querySelector('.minhaClasse');
  • querySelectorAll(seletor): Seleciona todos os elementos que correspondem ao seletor CSS especificado. Retorna uma NodeList. Exemplo:
    let elementos = document.querySelectorAll('.minhaClasse');
